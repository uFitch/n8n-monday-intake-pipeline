# Monday.com Board Structure

## Board 1: Intake Review

Groups: New Submissions → Under Review → Approved → Rejected

Columns: Company Name, Contact Email, Project Title, Description,
Development Stage, Budget Range, Timeline, Technical Requirements,
Files, Review Status, Reviewer, Submitted At

## Board 2: Project Pipeline

Groups (phases): Discovery → Design & Engineering → Prototyping →
Testing & QA → Production → Completed

Columns: Company Name, Contact Email, Phase, Assigned To, Priority,
Due Date, Progress, Notes, Files

## Workflow

Form Submit → Intake Review "New Submissions"
→ Team reviews → Status changed to "Approved"
→ Auto-creates item in Pipeline "Discovery" group
→ Team progresses through phases manually
